---
title: '空标签 (Fragments)'
---

`html!` 宏可以直接接受多个根节点。在根层级**不需要**使用 fragment 包装 (`<>...</>`)：

```rust
use yew::prelude::*;

html! {
    <div></div>
    <p></p>
};
```

## 带 key 的 fragment

fragment 语法仅剩的合理用途是**带 key 的** fragment。

```rust
use yew::prelude::*;

html! {
    // .. siblings ..
    <key="group-a">
        <span>{"a1"}</span>
        <span>{"a2"}</span>
    </>
    // .. siblings ..
};
```

在某些场景下，带 key 的 fragment 有助于差分性能和节点持久化。
