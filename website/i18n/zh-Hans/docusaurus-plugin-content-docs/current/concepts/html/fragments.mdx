---
title: '空标签 (Fragments)'
---

import Tabs from '@theme/Tabs'
import TabItem from '@theme/TabItem'

`html!` 宏总是需要一个根节点。为了绕过这个限制，您可以使用一个“空标签”（也称为“fragments”）。

<Tabs>
<TabItem value="Valid" label="Valid">

```rust
use yew::prelude::*;

html! {
    <>
        <div></div>
        <p></p>
    </>
};

```

</TabItem>

<TabItem value="Invalid" label="Invalid">

```rust, compile_fail
use yew::prelude::*;

// 错误：只允许一个根 HTML 元素

html! {
    <div></div>
    <p></p>
};

```

</TabItem>
</Tabs>
