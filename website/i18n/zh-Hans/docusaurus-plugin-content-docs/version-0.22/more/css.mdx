一个关于如何最好地将 CSS 支持集成到 Yew 中的讨论可以在这里找到：[https://github.com/yewstack/yew/issues/533](https://github.com/yewstack/yew/issues/533)

这里包含了很多关于如何最好地将 CSS 支持集成到 Yew 中的讨论。

目前，我们采用的方法是鼓励开发者在采用最流行的系统之前构建许多系统。

社区目前正在开发几个项目，以便为项目添加样式。以下是其中的一些：

#### 组件库

-   [yew_styles](https://github.com/spielrs/yew_styles) - 一个没有任何 JavaScript 依赖的 Yew 样式框架。
-   [yew-mdc](https://github.com/Follpvosten/yew-mdc) - Material Design 组件。
-   [muicss-yew](https://github.com/AlephAlpha/muicss-yew) - MUI CSS 组件。
-   [Yewtify](https://github.com/yewstack/yewtify) – 在 Yew 中实现 Vuetify 框架提供的功能。

#### 样式解决方案

-   [stylist](https://github.com/futursolo/stylist-rs) - 用于 WebAssembly 应用程序的 CSS-in-Rust 样式解决方案。
-   [tailwind-css](https://github.com/thedodd/trunk/tree/master/examples/yew-tailwindcss) - Tailwind 实用类。

:::important 改进文档
如果您正在开发一个为 Yew 添加样式的项目，请提交一个 PR 将自己添加到这个列表中！
:::
