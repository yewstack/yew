## 什麼是不可變型別？

這些類型是您可以實例化但永遠不會更改值的類型。為了更新值，您必須實例化一個新值。

## 為什麼使用不可變型別？

與 React 一樣，屬性是從祖先傳播到子代的。這意味著屬性在每個元件更新時必須存在。這就是為什麼屬性應該——理想情況下——很容易克隆。為了實現這一點，我們通常將事物包裝在 `Rc` 中。

不可變類型非常適合保存屬性的值，因為它們可以在從組件傳遞到組件時以很低的成本克隆。
